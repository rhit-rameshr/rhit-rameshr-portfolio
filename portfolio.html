Python Soccer Shootout

I developed a Python-based soccer shootout game featuring interactive graphics, real-time arrow and player movement, shooting mechanics, and score tracking to enhance the overall gameplay experience. I applied object-oriented programming principles to organize the game's core components—such as player, ball, and goalkeeper—into separate classes, ensuring efficient code structure and functionality.

Java Flappy Bird

I developed an astronaut-themed Flappy Bird game in Java, incorporating custom game logic, physics, and collision detection, all built using object-oriented principles for efficient code structure. Working in a team of three, I contributed to the design and implementation of the game, helping to develop unique graphics, gravity-based mechanics, and intuitive controls to enhance the overall player experience.
